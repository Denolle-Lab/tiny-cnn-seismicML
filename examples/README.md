# Examples

This directory contains basic usage examples and utility scripts.

## Files

### `usage_example.py`
Basic Python script demonstrating how to use the CNN models programmatically.

**Purpose**: Quick reference for using the model in Python scripts
**Usage**: `python usage_example.py`

### `test_workflow.py`
Test script for validating the complete workflow.

**Purpose**: Testing and validation
**Usage**: `python test_workflow.py`

## Main Workflow Notebooks

For the complete workflow with Jupyter notebooks, see the `../notebooks/` directory:

- **Data Exploration**: `notebooks/01_data_exploration/`
- **Data Labeling**: `notebooks/02_labeling/`
- **Model Training**: `notebooks/03_training/`
- **Inference**: `notebooks/04_inference/`

See `../notebooks/README.md` for detailed documentation.
